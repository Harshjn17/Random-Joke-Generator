😂 Random Joke Generator

A simple JavaScript project that fetches and displays random jokes using a public API.

🚀 Features

- Fetches a random joke with a single click
- Uses the Fetch API
- Built with "async/await"
- Handles errors using "try...catch"
- Displays a loading message while fetching data
- Clean and responsive user interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

📂 Project Structure

├── index.html
├── style.css
├── script.js
└── README.md

▶️ How to Run

1. Clone this repository.
2. Open the project folder.
3. Open "index.html" in your browser.
4. Click the Get Joke button to fetch a random joke.
